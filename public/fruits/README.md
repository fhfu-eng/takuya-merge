## 配置先

スイカゲームのプロジェクト内 `public/fruits/` 配下に配置してください。

```
public/
  fruits/
    01_cherry.png
    02_strawberry.png
    ...
    11_watermelon.png
```

## 仕様

- 形式: PNG (透過チャンネル付き)
- サイズ: 300×300px (全画像統一)
- 背景: 透過
- 配置: 顔が画像中心
